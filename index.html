What is Centauri?

Centauri is a multifunctional, and multipurpose bot, with its own economy, and many categories and commands!

Main Commands

To view all commands, type &help.
To change the prefix for the server, type &setprefix <prefix>.
Please note to not write the <> symbols when using the command.
